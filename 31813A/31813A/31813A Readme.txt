Part number 31813 Rev A

Four layer board .062 inches thick

Rectangular PCB dimensions: 6.360 inches in X and 4.630 inches in Y.  All drill files and gerber files are in inches.

files:

31813A.cmp   = Top side of board, layer 1, Component side etch
31813A.gpl   = layer 2, ground plane
31813A.ppl   = layer 3, power plane
31813A.sol   = Bottom of board, layer 4, Solder side etchs
31813A.plc   = Component side silk screen
31813A.pls   = Solder side silk screen
31813A.stc   = Solder mask component side
31813A.sts   = Solder mask solder side
31813A.drd   = Excellon drill file
31813A.oln   = Outline file

All files are viewed from the component side and are oriented with 0,0 being the lower left corner of the board.  
Positive X is going right from the origin and positive Y is going up from the origin.

None of the files are mirrored and should stack properly in a Gerber viewer program.

Solder mask is green, silkscreen is white.
Solder mask both sides
Silk screen both sides

1 oz copper

Material FR4 or equivalent
